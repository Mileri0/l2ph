Изменения в плагинах по сравнению с 3.5

Изменены:

TOnPacket = procedure(const cnt: Cardinal; const fromServer: Boolean; const connectionname:string; var packet : tpacket); stdcall;
теперь передаеться имя соединения.

Убрано поле UserFuncs с структуры TpluginStruct, связано с тем что это теперь возлагаеться на сам paxCompiller
Добавить функцию можно используя модуль PaxRegister из paxCompiller. Регистрацию производить все в том же OnRefreshPrecompile.
Модуль можно найти в файловом архиве програмы.
